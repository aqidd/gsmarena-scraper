# gsmarena_scraper

This script extract the mobile specs from all the phones available in gsmarena.com to a csv file.

## Pre-requisites

- pandas
- beautifulsoup

## Installation

```
pipenv install
```

## Usage

```
pipenv run python gsmarena_scraper.py
```

## Help

```
pipenv run python gsmarena_scraper.py -h
```
